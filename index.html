¿Qué es lo que hago?

Aunque mis últimos proyectos son páginas web estáticas o con funciones de pago y administradas por un panel creado a la medida, o en wordpress depende el requerimiento del cliente. La mayoría del dia paso desarrollando sistemas ERP para la empresa que trabajo ElementRH. Donde he desarrollado algunos sistemas, el más importante es el de la empresa donde se capturas todos los empleados subcontratados, se hacen movimientos para el IMSS, se piden requisiciones de pago y estas son aprobadas por otros usuarios, se envían confirmaciones por email, entre otros procesos que realiza el sistema.

Programación de sistemas

Desarrollo sistemas en base a los requerimientos del cliente, por lo regular siempre sugiero que sea un sistema online donde no dependa de un sistema operativo en especifico para funcionar. Por lo regular utilizo PHP como backend MySql como base de datos y js, css y html para el frontend.

Páginas web

Creación y desarrollo de páginas web, donde por lo regular utilizo PHP como backend MySql como base de datos y js, css y html para el frontend. Siempre se sugiere usar una plantilla para la página web pero si el cliente ya tiene un diseño sin problema se crea el diseño en web.

Aplicaciones de consola para windows

Ha surgido la necesidad de crear aplicaciones de consola en C# para hacer copia de archivos a un servidor remoto, validar diferencias entre bases de datos y hacer backups de bases de datos.

Aplicaciones para teléfonos moviles

Desarrollo de aplicaciones móviles multiplataforma con Ionic y Angular. En este momento estoy desarrollando una app de una purificadora de agua.

Soporte técnico

En mi trabajo también doy mucho soporte técnico a nuestros compañeros, ya sea con problemas técnicos en sus equipos que se pueden resolver vía remota con teamviewer o con problemas en los sistemas que manejamos. También a mis clientes les doy soporte para configuración de emails y capacitación para el uso de plataformas.
